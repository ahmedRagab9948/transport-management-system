# Transport Management System (TMS)

A full-stack enterprise transportation management system for logistics and transportation companies.

The system manages:
- Vehicles
- Drivers
- Trips
- Clients
- Employees
- Permissions
- Reports

Vehicle Types:
- Trailer Trucks
- Jumbo Trucks

Trip Types:
- Per Trip
- Monthly Contract

Main Features:
- OTP Authentication
- Role-based access control
- Real-time trip tracking
- Vehicle assignment
- Driver assignment
- Audit logs
